# 项目书截图清单

以下截图可用于课程项目书、答辩 PPT 或验收材料。

## 一、系统页面截图

1. 登录页截图
   - 展示系统名称、登录表单、测试账号入口。

2. 控制台首页截图
   - 展示 Bug 总数、今日新增、未处理、已修复、已关闭、严重 Bug 数量。
   - 展示状态统计图和严重程度统计图。

3. Bug 列表页截图
   - 展示 Bug 表格。
   - 展示标题搜索、状态筛选、严重程度筛选。
   - 展示详情、编辑、状态、转交、删除等按钮。

4. 新增 Bug 页面截图
   - 展示标题、所属模块、严重程度、描述、复现步骤、截图地址、处理人字段。

5. Bug 详情页截图
   - 展示 Bug 基础信息、问题描述、复现步骤、截图地址。
   - 展示状态流转提示：已提交 → 已修复 → 已关闭。

6. 状态流转操作截图
   - 展示修改状态弹窗。
   - 建议截图开发人员把 Bug 改为“已修复”，或测试人员把 Bug 改为“已关闭”。

7. 转交处理人截图
   - 展示转交弹窗。
   - 展示选择新处理人和填写备注。

8. 操作记录截图
   - 展示 Bug 详情页下方时间线。
   - 体现新增、编辑、修改状态、转交等记录。

9. 统计分析页截图
   - 展示统计卡片和 ECharts 图表。

10. 用户管理页截图
    - 使用管理员账号登录后截图。
    - 展示 admin、tester、developer 三个用户。

## 二、部署和运行截图

1. Docker 构建启动截图

```bash
docker compose up -d --build
```

2. Docker 容器运行截图

```bash
docker compose ps
```

3. Docker 日志截图

```bash
docker compose logs -f
```

4. openEuler 系统信息截图

```bash
cat /etc/os-release
uname -m
```

重点展示 `aarch64`。

5. Docker 版本截图

```bash
docker version
docker compose version
```

6. 防火墙端口截图

```bash
sudo firewall-cmd --list-ports
```

重点展示 `8080/tcp`。

7. 服务器浏览器访问截图

```text
http://服务器IP:8080
```

## 三、推荐截图顺序

建议项目书按以下顺序放图：

1. 系统登录页
2. 控制台首页
3. Bug 列表
4. 新增 Bug
5. Bug 详情
6. 状态流转
7. 操作记录
8. 统计分析
9. Docker 容器运行
10. 服务器访问结果
