# Frontend

Vue 3 + Vite + Element Plus 前端应用。

## 已实现页面

- 登录页
- 后台主布局
- 控制台首页
- Bug 列表页
- 新增 / 编辑 Bug 页面
- Bug 详情页
- 统计分析页
- 用户列表页

## 已实现能力

- Axios 请求封装
- Token 本地保存
- 路由守卫
- 角色菜单控制
- 不同角色操作按钮控制
- ECharts 统计图表
- React Bits 风格参考的动态网格背景、玻璃卡片和轻量交互动效

## 本地运行

```bash
npm install
npm run dev
```

开发环境默认通过 Vite 代理访问后端：

```text
/api -> http://localhost:8081
```

## 构建

```bash
npm run build
```
